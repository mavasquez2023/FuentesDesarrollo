Barra de herramientas de AFP Plug-in Viewer

Seleccione el icono de Zoom para visualizar el recuadro de diálogo Zoom . Utilice el recuadro de diálogo Zoom para cambiar el factor de zoom de los documentos visualizados.

Seleccione el icono de Rotación de página para visualizar el recuadro de diálogo Rotación de página . Utilice el recuadro de diálogo Rotación de página para cambiar el modo en que Viewer visualiza la información. Los valores válidos son Definir por archivo , 90 grados en sentido horario, 90 grados en sentido antihorario y 180 grados.

Seleccione el icono de Restablecer vista para cambiar el zoom y la orientación a los valores por omisión para el documento.

Seleccione el icono de Ocultar o mostrar imagen para ocultar o mostrar imágenes del documento.

Seleccione un área rectangular del documento. Después de pulsar este mapa de bits de la barra de herramientas, utilice el ratón para seleccionar un área del documento. Plug-in dibuja un rectángulo alrededor del área seleccionada. A continuación, pulse el mapa de bits de la barra de herramientas Copiar selección al portapapeles. Plug-in copia el área seleccionada al portapapeles como un mapa de bits.

Seleccione texto del documento. Después de pulsar este mapa de bits de la barra de herramientas, utilice el ratón para seleccionar una o más líneas del documento. Plug-in resalta el texto en el área seleccionada. A continuación, pulse el mapa de bits de la barra de herramientas Copiar selección al portapapeles. Plug-in copia las palabras del área seleccionada en el portapapeles como texto.

Copie la selección al portapapeles. Después de copiar una área o texto del documento, pulse este mapa de bits de la barra de herramientas para copiar la selección en el portapapeles.

Seleccione el icono Ir a para visualizar la ventana Ir a. Utilice la ventana Ir a para seleccionar una página del documento para visualizarla.

Seleccione el icono Buscar para visualizar el recuadro de diálogo Buscar. Utilice el recuadro de diálogo Buscar para buscar y resaltar una palabra o serie de texto.

Seleccione el icono Buscar siguiente para visualizar la siguiente instancia de la serie de texto que se busca mediante el recuadro de diálogo Buscar.

Seleccione el icono Primera página para visualizar la primera página del documento.

Seleccione el icono Página anterior para visualizar la página anterior del documento.

Seleccione la Página siguiente para visualizar la página siguiente del documento.

Seleccione el icono Última página para visualizar la última página del documento.

Seleccione el botón de la barra de herramientas Imprimir para visualizar el recuadro de diálogo Imprimir e imprimir el documento que está visualizando.

Seleccione el icono Ayuda para acceder al índice de ayuda de Viewer.

 

Para obtener más ayuda acerca de AFP Plug-in Viewer, utilice la Ayuda entregada directamente por el Plug-in. Podrá obtener información detallada de cada posibilidad entregada por las distintas opciones de la barra de herramientas descrita.
