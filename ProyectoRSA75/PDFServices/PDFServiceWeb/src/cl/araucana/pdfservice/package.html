Core classes and abstractions to implement {@link cl.araucana.pdfservice.PDFService}. The main objetive is to provide a general model to produce and to consume PDF documents under a pipeline execution schema implemented with {@link cl.araucana.pdfservice.PDFProcess} (a coordinated collection of customizable thread workers).

This service is designed to be integratable to both standalone or J2EE applications and based on MVC and Producer/Consumer design patterns and Fast PDF Generator (FPG) framework.

@since 1.4
