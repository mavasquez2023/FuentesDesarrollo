Provides a complete implementation of a PDF Publisher that is integrated to the framework FPG. Its main design objectives are:

  • To be simple.
  • To preserve document layout produced by FPG to provide the best performance/cost relation.
  • To be neutral to the operating system platform.
  • To provide a relational documental repository.
  • To support different RDBMS systems.
  • To be executable in both standalone mode and J2EE platform.

{@link cl.araucana.pdf.publishers.implementations.fpg.FPGIntegratedPDFPublisherSPI} class documentation shows a summary of documental repository model used by this PDF Publisher.
